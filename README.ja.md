# ar-snow

3Dの雪の結晶のアニメーションを表示するAR/VRアプリです。WebXRに対応しており、PC、スマートフォン、VRヘッドセット等で体験できます。

## デモ
https://code4fukui.github.io/ar-snow/

## 機能
- 3D雪の演出
- WebXR対応（AR/VR）

## 必要環境
- three.js

## 使い方
WebXRに対応したブラウザで[デモ](https://code4fukui.github.io/ar-snow/)を開くと、ARやVRで雪の演出を体験できます。
