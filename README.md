## Dependencies

- [three.js](https://threejs.org/)

## Features

- Displays falling snow particles in Augmented Reality (AR) or Virtual Reality (VR)
- Uses the WebXR API for AR/VR support
- Implements a clipping plane to create a snow globe effect
- Allows user to start and stop the AR/VR experience via a button

## Usage

To run the project, simply open the `index.html` or `hex.html` file in a web browser. The project uses the WebXR API, so it requires a WebXR-enabled browser and device.
